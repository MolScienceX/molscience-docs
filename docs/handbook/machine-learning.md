# 机器学习基础

本页介绍 MolScience 中常见的机器学习基础概念，帮助新成员理解数据如何进入模型，以及如何判断模型结果是否可靠。

## 重点概念

- 数据集：用于训练、验证或测试模型的数据集合。
- 特征：模型输入的信息，例如分子描述符、指纹、结构图或实验条件。
- 标签：模型要预测的目标，例如性质数值、类别或实验结果。
- 训练与测试：训练集用于拟合模型，测试集用于估计泛化能力。
- 模型评估：根据任务类型选择合适指标，例如误差、准确率、召回率或相关系数。
- 过拟合与泛化：模型在训练集表现很好，但在新数据上表现差，通常说明泛化不足。

## 分子机器学习中的注意事项

- 训练集和测试集不能包含重复或高度泄漏的数据。
- 特征生成方式要可复现，并记录软件版本或参数。
- 数据量较小时，评估结果可能不稳定，需要谨慎解释。
- 模型结论应回到化学语境中检查，避免只看指标。

## 新成员可以先练习

1. 阅读一个已有数据集说明，找出特征、标签和单位。
2. 复现一个简单模型训练流程。
3. 对比训练集和测试集指标，判断是否可能过拟合。
4. 把复现步骤和遇到的问题写入文档或 issue。
