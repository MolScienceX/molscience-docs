# 化学基础

本页整理参与 MolScience 项目时常用的化学基础概念。它不是完整教材，而是帮助新成员读懂数据、文献和模型任务的入门清单。

## 重点主题

- 分子结构：原子、化学键、构型、构象和立体化学。
- 官能团：常见官能团及其对性质和反应性的影响。
- 物理化学概念：溶解度、酸碱性、极性、热力学和动力学。
- 化学性质：稳定性、反应条件、实验测量值和计算性质。
- 数据可靠性：实验条件、测量误差、单位、来源和适用范围。

## 阅读数据时要注意

- 同一个性质可能受温度、溶剂、pH 或测量方法影响。
- 分子名称、结构式、SMILES 和 InChI 等标识需要相互核对。
- 计算值和实验值不能混用为同一种证据，除非文档中明确说明。
- 异常值不一定是错误，但必须记录检查过程和可能原因。

## 建议补充的能力

- 能够识别常见分子表示方式。
- 能够说明一个性质的单位和实验条件。
- 能够根据来源判断数据是否适合进入数据库或模型训练集。
