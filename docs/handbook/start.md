# 从这里开始

本页帮助新成员快速了解 MolScience 平台，并完成加入初期最重要的准备工作。

## 平台目标

MolScience 关注化学、数据和机器学习在分子科学问题中的结合。平台文档的目标是把项目背景、数据规范、协作流程和学习材料沉淀下来，让成员可以稳定地交接和复现工作。

## 第一周建议

1. 阅读新人手册首页和常见问题，了解整体结构。
2. 熟悉 GitHub 基本操作，确认自己能提交 issue 或 pull request。
3. 选择一个小任务开始，例如修正文档、补充术语、整理一条数据来源或复现一个简单脚本。
4. 在任务中记录不清楚的地方，并优先把这些问题转化为文档改进。

## 参与方式

- 文档方向：整理说明、翻译页面、补充示例和维护术语一致性。
- 数据方向：收集数据来源、检查字段、统一单位和记录数据质量问题。
- 化学方向：解释分子结构、性质、实验条件和化学语境。
- 机器学习方向：准备数据集、提取特征、训练模型和评估结果。

## 基本原则

- 先读已有文档，再提出修改。
- 任何数据都要记录来源、单位和不确定性。
- 小步提交，保持 pull request 聚焦。
- 遇到不确定内容时，优先提 issue 或在文档中标注待确认。
