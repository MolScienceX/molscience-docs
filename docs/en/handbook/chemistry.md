# Chemistry Basics

This page collects basic chemistry concepts commonly used in MolScience. It is not a complete textbook; it is an onboarding checklist for reading data, literature, and model tasks.

## Key topics

- Molecular structure: atoms, bonds, configuration, conformation, and stereochemistry.
- Functional groups: common groups and how they affect properties and reactivity.
- Physical chemistry concepts: solubility, acidity/basicity, polarity, thermodynamics, and kinetics.
- Chemical properties: stability, reaction conditions, experimental measurements, and computed properties.
- Data reliability: experimental conditions, measurement error, units, sources, and scope of use.

## When reading data

- The same property may depend on temperature, solvent, pH, or measurement method.
- Names, structural formulas, SMILES, InChI, and other identifiers should be cross-checked.
- Computed values and experimental values should not be treated as the same type of evidence unless the documentation says so.
- Outliers are not always errors, but the checking process and possible reasons should be recorded.

## Recommended skills

- Recognize common molecular representations.
- Explain the unit and experimental condition of a property.
- Judge whether a source is suitable for the database or a model training set.
