# Start Here

This page introduces how to get started with MolScience.

## Platform goals

MolScience focuses on the intersection of chemistry, data, and machine learning for molecular science problems. The documentation keeps project background, data standards, collaboration workflows, and learning materials in one place so that work can be handed over and reproduced reliably.

## First-week checklist

1. Read the handbook overview and FAQ to understand the documentation structure.
2. Learn the basic GitHub workflow and confirm that you can open an issue or pull request.
3. Pick a small first task, such as fixing documentation, adding terminology, organizing one data source, or reproducing a simple script.
4. Record unclear points during the task and turn them into documentation improvements when possible.

## Ways to participate

- Documentation: write explanations, translate pages, add examples, and keep terminology consistent.
- Data: collect data sources, check fields, standardize units, and record data quality issues.
- Chemistry: explain molecular structures, properties, experimental conditions, and chemical context.
- Machine learning: prepare datasets, extract features, train models, and evaluate results.

## Basic principles

- Read existing documentation before proposing changes.
- Record the source, unit, and uncertainty for every data item.
- Make small commits and keep each pull request focused.
- When something is uncertain, open an issue or mark it clearly in the documentation.
