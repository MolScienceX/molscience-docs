# MolScience Handbook

MolScience Handbook is the entry point for new members.

It explains the background, collaboration workflow, basic knowledge, and common questions needed to participate in MolScience projects. Whether you start with documentation, data, experimental records, code, or machine learning tasks, use this handbook to build shared context first.

## Suggested reading order

1. [Start Here](./start): Learn the platform goals, role expectations, and first-week checklist.
2. [GitHub Guide](./github): Learn the basic workflow for issues, branches, commits, and pull requests.
3. [Database Guide](./database): Understand data fields, source records, and quality checks.
4. [Chemistry Basics](./chemistry): Build context around molecular structure, properties, and data reliability.
5. [Machine Learning Basics](./machine-learning): Understand datasets, features, labels, and model evaluation.
6. [FAQ](./faq): Review common questions from new members.

## How new members can contribute

- Improve or translate documentation and clarify confusing explanations.
- Organize literature, experimental records, or public data sources.
- Check database fields, units, missing values, and data provenance.
- Contribute to code, scripts, model training, or result reproduction.
- Open issues for questions and record improvements found during onboarding.
